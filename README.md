# dotfiles

Hyprland-based Wayland dotfiles managed with [GNU Stow](https://www.gnu.org/software/stow/).

## Tools

| Tool | Purpose |
|------|---------|
| [Hyprland](https://hyprland.org) | Wayland compositor |
| [Kitty](https://sw.kovidgoyal.net/kitty) | Terminal emulator |
| [Neovim](https://neovim.io) | Editor |
| [tmux](https://github.com/tmux/tmux) | Terminal multiplexer |
| [Starship](https://starship.rs) | Shell prompt |
| [Zoxide](https://github.com/ajeetdsouza/zoxide) | Smart `cd` |
| [Waybar](https://github.com/Alexays/Waybar) | Status bar |
| [Wofi](https://hg.sr.ht/~scoopta/wofi) | App launcher |
| [Mako](https://github.com/emersion/mako) | Notifications |
| [Hyprlock](https://github.com/hyprwm/hyprlock) | Lock screen |
| [Hypridle](https://github.com/hyprwm/hypridle) | Idle daemon |
| [Waypaper](https://github.com/anufrievroman/waypaper) | Wallpaper manager |
| [Hyprpaper](https://github.com/hyprwm/hyprpaper) | Wallpaper backend |
| [zsh-autosuggestions](https://github.com/zsh-users/zsh-autosuggestions) | Shell autosuggestions |
| [zsh-syntax-highlighting](https://github.com/zsh-users/zsh-syntax-highlighting) | Shell syntax highlighting |

## Structure

| Folder | Config |
|--------|--------|
| `hypr/` | Hyprland, Hyprlock, Hypridle, Hyprpaper |
| `kitty/` | Kitty terminal |
| `nvim/` | Neovim |
| `tmux/` | tmux |
| `starship/` | Starship prompt |
| `waybar/` | Waybar status bar |
| `wofi/` | Wofi launcher |
| `mako/` | Mako notifications |
| `zshrc/` | Zsh config |
| `backgrounds/` | Wallpapers |
| `sessions/` | Wayland session desktop entry |
